## เว็บไซต์
เป็นหน้าเว็บ static ไฟล์เดียว — เปิด `index.html` ได้ทันที ไม่ต้อง build

## Deploy บน Vercel
1. Import repo นี้ใน Vercel
2. Framework Preset: **Other** (ไม่ต้องตั้ง build command)
3. Output/Root Directory: `/` (root)
4. Deploy ได้เลย — Vercel จะเสิร์ฟ `index.html` โดยอัตโนมัติ

## โครงสร้าง
- `index.html` — หน้าเว็บทั้งหมด (HTML/CSS/JS ในไฟล์เดียว)
- `assets/` — รูปโลโก้และรูปสินค้าจริง
